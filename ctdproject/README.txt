# Tongue Tied

## Getting Started
<ol>
    <li>Download the files from git hub</li>
<li>In Tongue_Tied.py change the gif file locations to your local file locations</li>
    <ol>
    <li>Do take note that some time if you do not put full file location it may not be able to find the file </li>
    </ol>
<li>Run the File how you may like in a IDE or compiler to play the game</li>
    <ol>
    <li>An example you can start the terminal and cd into your directory which contains the python files.</li>
    <li> type the command  " python3 Tongue_Tied.py " to run the code and start to play the game</li>
    </ol>
</ol>

## Keys Required for the Gameplay
capital letter or lowercase letter doesn't matter \
"k" key to start the game\
"q" key to quit the game\
"w","a","s","d" key is to control the movement of player 1\
The arrows key is to control the movement of player 2

### Dependencies
- Python3
- math library
- turtle library
- time library
- Windows 10, 11
